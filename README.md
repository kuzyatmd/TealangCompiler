## Getting Started

To get started download source code, `cd TealangCompiler`, `source ./environment.sh`.

## Debug Build
Currently only Debug build is supported. I recommend using commands from `./environment.sh` to control current state.

- `tea-rebuild` to clear last build and build current code in debug to `./build`.
- `tea` to launch current build with arguments from `./params` prepended to passed args (`tea your arguments` -> `{params} your arguments`)
